## 简介
   通过华为hQUIC Kit示例代码，您将体验到如何使用QUIC协议通过hQUIC Kit与后端服务器交互。

## 开发准备
   1. 检查Android Studio开发环境是否准备就绪。
   2. 打开示例项目，修改HQUICActivity.java文件中的URL。该URL必须是GET请求的URL，并且支持QUIC协议。
   3. 在你的安卓设备上运行示例代码。

## 环境要求
   推荐Android SDK版本号为19或以上，JDK版本号为1.7或以上。
